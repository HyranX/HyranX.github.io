作为人类的挫败感

发表于 2017-08-31   |   分类于 Logic   |  

  1. 越限制越自由
    通常来说，如上句式都讨厌：它刻意隐去必要陈述，只为造成剩余部分严格违反直觉。读者越是对它迷惑不解，作者解释起来越是胸有成竹娓娓道来。而此时，我是想说另一件事：如果一种乐器的表现力受到妥协，且如果这种妥协恰如听众所期望，甚至最关注，那么面对这类听众，操持这种乐器创作的艺术家才可最大限度释放表达自由。
    拨弦古钢琴的琴键推动连杆带动羽毛管拨弦发声。无论演奏者使用多大力气，最终也只传递羽毛管的硬度所对应的拨动响度。这相比钢琴受到了天大的限制。但当使用这种乐器演奏复调音乐时，奇妙的作用产生了。我们通常希望在复调音乐中体验到的复杂交织，被拨弦古钢琴加强了：它没法像钢琴演奏者那样避开繁复，只把自己挑选旋律弹重给人听：复杂必须被呈现。

  2. 作为人类的挫败感
    而此时，我只是想说另一件事：当我们自身作为人类的个体，轻易探索即可触及人类引以为豪的理解能力的极限时，所体验到的挫败感让你不想起床。
    尝试理解复调音乐，就是这样的时刻。听力尚可的人，稍作努力，就可以同时追踪复调音乐中的两条独立线索，但线索的组合会创生新的感受，此时「A+B」≠「A」+「B」，刚刚我只是做到了同时追踪「A」和「B」，但是无论如何，普通人类似乎永远无法做到同时追踪「A」和「A+B」。当我们尝试这样做的时候，理解能力的极限就直白的摆在眼前，它本身并不复杂，使用Synthesia播放巴赫，你就能理解这种不复杂，但就像2毛钱的超市塑料袋也能让你窒息一样，面对「A」和「A+B」我希望你和我一样无可奈何。
    这种挫败感受似乎仅能通过音乐获得，因为其他方式总能通过事后分析获得全部需要被理解的信息。例如你当然可以编写剧本让5个人同时开口争辩，这种人造复杂经不起事后逐字分析。但音乐体验只能发生在进行时，毫无「事后」可言。我希望你也一样无可奈何。

  3. 沉迷拨弦古钢琴并不是一场常见的虐恋
    沉迷拨弦古钢琴并不仅仅是在主动寻求和体验受限，它通过逼近人类理解极限，配合冰冷刻板的对位法则，传递一种丰富而罕见的追问：我们在巴洛克音乐中体验到的情感究竟来自哪里？既不是力求抹平差异的羽毛管拨弦结构，也不是来自于墨守森严的规则（总是能在规则内创造无穷无尽的烂旋律）。内心情感似乎是博尔赫斯酒馆中蔓延至地平线的宏伟酒架上色彩斑斓的预调鸡尾酒：没有所谓新情绪的创造，只有预先调配好的情绪被触发。拨弦古钢琴配合对位法就像一把刻意简化的钥匙，逼你相信，此刻所目睹、所体验，并非来自眼前纷繁世界，而是来自紧锁的心智大门背后，人类于创生就携带，只能调用缺无法增减和改变的理解力仓库。

  4. ……
    而躺在床上的我，此时真希望自己在想的是另一件事。
